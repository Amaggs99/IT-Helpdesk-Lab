\# Overview



This repository documents realistic IT Help Desk scenarios performed within a Windows Server 2022 Active Directory lab environment.



Each scenario simulates a real support ticket and includes:



\- Step-by-step resolution

\- PowerShell commands

\- GUI administration

\- Troubleshooting process

\- Screenshots

\- Documentation



The goal is to demonstrate practical Help Desk and Systems Administration skills commonly used in enterprise environments.



\---



\# Objectives



\- Simulate real-world Help Desk tickets

\- Perform common Active Directory administration tasks

\- Practice PowerShell administration

\- Document troubleshooting procedures

\- Build a professional GitHub portfolio

\- Develop repeatable IT support workflows
